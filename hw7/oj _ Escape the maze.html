Escape the maze

Description

you should implement code to find the shortest road to escape the maze.

You can use all algorithm mentioned in class.

Input

  • You will be given a m * n maze and each cell could be wall or road,

    '#' represent wall, ' ' represent the road you can walk, '*' represent the start point, '+' represent maze exit point.

  • You are required to print '.' over all your steps to escape the maze

  • Can't walk in diagonal direction (x+1, y+1) or (x-1, y-1)
  • Also you should handle maze which can find the road
  • To ensure only one solution, you should move vertical direction then move horizontal direction as far as you can when you have multi path, you can see it in example 1

Output

  • You should print your result to stdout
  • If there is no solution(there are many special situations), save "NO SOLUTION" (no space in front of that, one space between "NO" and "SOLUTION") in your output
  • At the end of your output, add your shortest step number, no need for no-solution case.

Sample Input 1 

#+ ##
#   #
#  *#
#####

Sample Output 1

#..##
# ..#
#  .#
#####
4

Sample Input 2 

######
#    +
#  # #
## # #
#* # #
######

Sample Output 2

######
# ....
# .# #
##.# #
#..# #
######
7

Sample Input 3 

######
#  # +
#   ##
#    #
###  #
#*  ##
######

Sample Output 3

NO SOLUTION
